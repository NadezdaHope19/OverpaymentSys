# Документация проекта "Система учета переплат"
## Состав репозитория
* /diagrams/ - модели данных (AS IS, TO BE и ER-диаграммы)
* /technical-specs/ - техническое задание и требования к продукту
* /manual/ - руководство пользователя

Проект разработан для унификации работы отделов различных управлений по учету и работе с переплатами и автоматизации контрольных фуункций с целью повысить процент погашений задолженностей.

* проведен анализ сквозного бизнеса-процесса по работе с переплатами, в ходе которого выявлены узкие места, простои, отсутствие контрольных точек ([диаграмма AS IS](https://github.com/NadezdaHope19/OverpaymentSys/blob/main/diagrams/ASIS.png?raw=true))
* предложены и внедрены оптимизирующие мероприятия, позволяющие сократить количество итерраций, бумажного оборота между управлениями
* разработана новая модель бизнес-процесса с учетом внедрения программного комплекса, решающего проблему неэффективности межструктурного взаимодействия и отсутствия контроля ([диаграмма TO BE](https://github.com/NadezdaHope19/OverpaymentSys/blob/main/diagrams/TOBE.png?raw=true))
* написано [техническое задание](technical-specs/tech.md) на разработку ПО и построена [ER-диаграмма](https://github.com/user-attachments/assets/4de4b6cd-b04d-454f-91c9-60ee954c1023)
